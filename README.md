# Random Password Generator

**Автор:** Ярослав Новиков

Приложение для генерации случайных паролей с возможностью настройки и ведения истории.

## Особенности

- Выбор длины пароля (от 6 до 32)
- Настройка символов (буквы, цифры, спецсимволы)
- Автоматическое сохранение истории в JSON
- Интерфейс на Tkinter

## Использование

1. Выберите параметры пароля
2. Нажмите "Сгенерировать"
3. Используйте пароль и просмотрите историю в таблице

## Запуск

python main.py
